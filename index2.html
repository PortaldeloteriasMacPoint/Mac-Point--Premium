Bem-vindo! Você acessou a página protegida.

O que acontece agora: 1. Verificação de autenticação: Assim que a página index2.html é carregada, o código Firebase verifica se o usuário está autenticado. Se não estiver autenticado, o usuário será redirecionado automaticamente para a página bloqueado.html. 2. Proteção contra link direto: Se o usuário tentar acessar o link diretamente, sem estar autenticado, ele será redirecionado para a página de bloqueio, sem visualizar o conteúdo da página protegida. 3. Página de bloqueio: Caso o usuário não tenha permissão para acessar, ele será redirecionado para a página bloqueado.html. Esse arquivo pode ser configurado com uma simples mensagem informando que o acesso foi bloqueado ou que é necessário login. bloqueado.html (Página de Acesso Bloqueado) Acesso Bloqueado
